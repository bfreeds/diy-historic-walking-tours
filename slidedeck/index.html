In this workshop we will:

  • Learn the basics of web mapping using Leaflet.js
  • Learn to find and use maps from NYPL's collections
  • Create a geographic dataset of historic places
  • Create an interactive historic walking tour!

  • Raise your hand if you have used Leaflet before. People with your hands not raised - look around, these folks will be helpful if you get stuck along the way.
